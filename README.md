# CSDN评论区抽奖系统

一个基于Flask的CSDN评论区抽奖工具，支持直接从CSDN文章获取评论并进行随机抽奖。

## 功能特性

- **直接抽奖**：不需要先获取评论，可以直接点击"开始抽奖"按钮进行抽奖
- **折叠评论**：支持选择是否包含折叠评论(默认勾选)
- **回复评论**：支持选择是否包含回复评论(默认不勾选)
- **评论去重**：自动去重，同一个用户只保留一条评论
- **用户跳转**：中奖用户名称可点击跳转到其CSDN主页
- **响应式设计**：界面美观，操作简单

## 使用方法

1. **启动应用**

```bash
python run.py
```

2. **访问系统**

   打开浏览器，访问 `http://127.0.0.1:8100`

3. **使用步骤**

   - 在"CSDN文章链接"输入框中输入完整的CSDN文章链接
   - 选择是否包含折叠评论（默认选中）
   - 选择是否包含回复评论（默认不选中）
   - 设置中奖人数
   - 点击"开始抽奖"按钮进行抽奖，或点击"获取评论"按钮查看所有评论
   - 抽奖结果会显示在页面下方，中奖用户名称可点击跳转到其CSDN主页

## 技术实现

- **后端**：使用Flask框架实现，提供两个API接口：
  - `/api/fetch_comments`：获取评论
  - `/api/run_lottery`：运行抽奖
- **前端**：使用HTML、CSS和JavaScript实现，页面设计美观，交互流畅
- **数据处理**：使用Python实现评论获取、去重和抽奖逻辑

## 注意事项

- 本工具仅用于学习和娱乐目的，请勿用于商业用途
- 抽奖结果完全随机，公平公正
- 由于CSDN API的限制，可能会出现获取评论失败的情况，请稍后重试
